# Dynamic Nand Gate (based on Visual Studio Community 2022, netcoreapp3.1)
Adds a NAND gate to Logic World that uses dynamic pegs, similar to the vanilla AND gate.

refer to https://github.com/FoxFireFive0135/Basic-Logic-World-Mod/blob/main/README.md when customizing this code for your own mods. It will help you.

This mod relys on Ecconia's mods, these 2 specifically:
https://github.com/Ecconia/Ecconia-LogicWorld-Mods/tree/master/EccsLogicWorldAPI
https://github.com/Ecconia/Ecconia-LogicWorld-Mods/tree/master/EccsGuiBuilder
